# book-tracker
Приложение для учета прочитанных книг. С помощью него можно добавлять, просматривать, удалять книги, а также смотреть статистику по книгам и авторам
Как запустить
1. Убедитесь, что ваш компьютер соответствует необходимым техныческим характеристикам(Python 3.6+)
2. Скачайте репозиторий
3. В терминале выполните python MAIN.py
Пример работы
 Трекер прочитанных книг
1. Добавить книгу
2. Показать все книги
3. Показать среднюю оценку
4. Статистика по авторам
5. Удалить книгу
6. Выход

Автор: <author>
Название: <name>
Оценка (1-5): <rate>
Дата прочтения (ГГГГ-ММ-ДД): <date>
